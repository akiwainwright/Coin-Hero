# Coin Hero

![Game](https://github.com/akiwainwright/My-Pacaman-like-game/blob/master/My%20game.png)

#### To start the game download the zip file, once downloaded extract the file to your computer and the open the file. Once you open the file at the top there is a debug file open that and run Pacman.exe file

### How to play

* Collect the coins and chest to increase gold
* Avoid Enemies
* Get to the ladder to complete the level
* Try and get as much gold as you can
* Once the game is started press c to open the game manual for more details
